# 🖼️ Item Image Popouts

> Players can finally click on item portraits and see the actual art.

By default in FoundryVTT, clicking on an item image opens the file picker — fine when the GM is editing the item, frustrating for a player who just wants to see what their occult tome actually looks like.

This module gives players a proper image popout while keeping the GM's file picker behavior where it matters.

## On the item sheet

Click any item's image (weapons, spells, books, skills, etc.) and the full illustration opens in a draggable, resizable popout window. **GMs still get the file picker** here, so they can keep editing item artwork in place.

## In the Gear & Cash tab

Click the small item icon next to any line in the Gear & Cash tab of the character sheet and the same popout opens. Works for everything shown in that tab: items, weapons, books, spells, armor, talents, and statuses. **GMs get the popout here too** — there's nothing to edit from a single line entry anyway.

![Item image popout from the Gear & Cash tab](../../images/Popup-Image.png)

Players have wanted this for a long time. Now they have it.
