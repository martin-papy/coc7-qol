# i18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extract all hardcoded user-visible strings into FoundryVTT translation JSON files, supporting English and French at launch.

**Architecture:** Flat `lang/en.json` and `lang/fr.json` files under a `COC7QOL.` prefix, registered in `module.json`. A `t()` / `tf()` helper pair exported from `utils.js` wraps `game.i18n.localize()` / `game.i18n.format()`. All call sites updated file by file. LLM prompt content (AI system/user prompts) is left hardcoded — it is not UI text.

**Tech Stack:** Vanilla JS ES modules, FoundryVTT v13 i18n API (`game.i18n.localize`, `game.i18n.format`), no build step.

## Timing constraint (read before implementing)

`game.i18n` is **not available at module load time**. Top-level `const` values and static class field initialisers run before the `init` hook fires. All `t()` / `tf()` calls must be inside methods, event handlers, or property getters that execute after `init`. The two main risks are:

1. **`CARD_TYPE_LABELS`** in `close-all-cards.js` — top-level const: convert values to i18n key strings, resolve via `t()` at use-time.
2. **`WEAPON_PROMPT_CONFIG` / `NPC_PROMPT_CONFIG`** in `dialog-injector.js` — top-level consts: convert translated string properties (`label`, `placeholder`, `prefillPrefix`, `extraHTML`) to getters.
3. **`DEFAULT_OPTIONS.window.title`** in both dialog classes — static field: remove the title from `DEFAULT_OPTIONS` and add a `get title()` getter on the class instead (standard ApplicationV2 pattern).

- [ ] **Step 1.4 — Add `t()` and `tf()` to `scripts/utils.js`**

Append to the end of `scripts/utils.js`:

```js
/**
 * Localise a static string by i18n key.
 * @param {string} key
 * @returns {string}
 */
export const t = key => game.i18n.localize(key)

/**
 * Localise a string with runtime data substitutions.
 * @param {string} key
 * @param {object} data
 * @returns {string}
 */
export const tf = (key, data) => game.i18n.format(key, data)
```

```js
function getOpenCards () {
  const open = [];
  for (const message of game.messages) {
    const load = message.flags?.CoC7?.load;
    if (load?.cardOpen === true) {
      const typeKey = CARD_TYPE_KEYS[load.as]
      open.push({
        messageId: message.id,
        type: typeKey ? t(typeKey) : t('COC7QOL.CloseAllCards.UnknownCardType'),
        actor: message.speaker?.alias ?? t('COC7QOL.CloseAllCards.UnknownActor'),
        timestamp: new Date(message.timestamp).toLocaleTimeString()
      });
    }
  }
  return open;
}
```
